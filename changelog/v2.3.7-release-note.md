# v2.3.7 Release Note

## Summary

v2.3.7 is a **recommended** release featuring ioctl improvements for better automation support, hot-update capability for producer keys, and a new IOSwarm coordinator.

## Key Features and Enhancements

### Hot-Update Producer-Key

Delegates can now update their `producerPrivKey` without restarting the node. This enables key rotation with zero downtime, improving operational security and flexibility for validator operators.

### IOSwarm

Introduces the IOSwarm coordinator for improved task scheduling and agent coordination, featuring multi-level task support (L1-L4) and shadow mode execution verification.

### ioctl Improvements

CLI tool enhancements for AI agents and automation workflows:

- **Non-interactive mode**: `--non-interactive` flag and `IOCTL_NON_INTERACTIVE=1` env var support
- **JSON output**: `messageType` serializes as string; added `--quiet/-q` flag
- **New commands**: `config dump`, `bc bucketlist all [OFFSET] [LIMIT]`
- **Batch queries**: `account balance addr1 addr2 addr3`
- **Wait for receipt**: `--wait/-w` flag for polling chain receipts

## Bug Fixes

- Fixed "out of range" errors
- Fixed `--wait` timeout handling
- Fixed SendAction/Execute nil response handling

## Upgrade Priority
